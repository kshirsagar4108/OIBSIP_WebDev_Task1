Landing Page

This project is a static landing page created as part of my internship web development tasks.

Description

The landing page represents a clean and modern company website layout. It includes a navigation bar, hero section with call-to-action buttons, company statistics, trusted brand logos, and a services/features section. The design focuses on layout structure, visual hierarchy, and UI styling.

Features
	•	Hero section with background image and overlay content
	•	Company statistics and recent project card
	•	Brand logos section
	•	Services/features grid layout

Technologies Used
	•	HTML5
	•	CSS3
	•	Remix Icon (for icons)

Purpose

This task was completed to practice HTML structure, CSS layout techniques (Flexbox), and real-world landing page design.
